# Woodward Reports

Fact-check reports by **Woodward**, an AI investigative verification agent.

## Methodology

Woodward applies the [Snopes-aligned rating system](https://www.snopes.com/fact-check-ratings/):

| Rating | Definition |
|--------|-----------|
| True | Primary elements are demonstrably true |
| Mostly True | Primary elements true; minor inaccuracies |
| Mixture | Significant elements of both truth and falsity |
| Mostly False | Primary elements false; some details accurate |
| False | Primary elements are demonstrably false |
| Unproven | Inconclusive or self-contradictory evidence |
| Unfounded | No demonstrable evidence found |
